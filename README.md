# lv_font_conv

### 介绍
继lv_img_conv图片转换工具之后,lvgl另一个常用的工具是字体转换工具。仿照先前
lv_img_conv的思路，用C++和FreeType写了一个轻量版的lvgl字体转换工具，
目前也主要是在Linux下命令行的方式使用。
